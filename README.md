# ATLAS CAPITAL - Commercial Real Estate Platform

This is a premium commercial real estate website for "ATLAS CAPITAL", featuring a modern, high-end design with advanced interactions and animations.

## Features

- **Splash Screen**: Animated loading sequence with building morphing effect.
- **Homepage**: Hero video/image, live market ticker, featured assets, market intelligence dashboard.
- **Listings Page**: Filterable property grid with advanced filtering options (Asset Class, Market, Size, etc.).
- **Property Detail Page**: Detailed view with hero carousel, investment highlights, rent roll, and interactive elements.
- **Insights Page**: Thought leadership platform with reports, podcasts, and market updates.
- **About Page**: Company history timeline, leadership team, and global footprint.
- **Contact Page**: Split-screen contact form with office locations.
- **Chatbot**: AI-powered assistant for investment inquiries.
- **404 Page**: Creative "missing building" concept.

## Tech Stack

- **React**: Frontend framework.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Framer Motion**: Animation library for complex transitions and interactions.
- **Lucide React**: Icon library.
- **React Router**: Client-side routing.

## Design

The design follows a "Neo-Brutalism meets Biophilic Design" aesthetic, using a dark mode palette with gold/copper accents and high-quality imagery.

## Usage

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
